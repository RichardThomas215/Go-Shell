# Go-Shell
Unix based shell built in Go-Lang

## Unix Commands Supported

  <br> <b>cd</b> </br>
  <b>rm</b> 
 <br> <b>ls</b> </br>
 <b>ls -al</b>
 <br> <b>pwd</b> </br>
 <b>touch</b>
   <br> <b>mkdir</b> </br>
   <b> whoami</b>

  ## Custom Commands
   <b>zip</b> - puts all files listed into a zip file called files.zip
    <li>ex: zip LS.go touch.go pwd.go </li>
